# mooling-skills

AI Agent skill documentation for my personal use.

# Usage

Except for README*.md, each `*.md` file represents a skill. You can ask the AI agent to convert it into formats usable by various CLI tools.

# For AI Agents

You need to convert the skill document into the correct format, rather than just copying and pasting the source file.

Normally, you should create a directory named after the skill, rename the skill document to SKILL.md, and place it in that directory.

Different tools manage skills in different ways, so you should handle it flexibly.
